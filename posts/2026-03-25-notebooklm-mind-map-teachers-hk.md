喺 NotebookLM 入面，**Mind Map**（心智圖）係 Studio 其中一種輸出：佢會根據你已上傳嘅 PDF、連結、文件等**來源**，整理出一幅**概念關係圖**，幫你由「一大堆字」快速跳到「邊個概念連住邊個」。

對老師嚟講，佢最實用嘅位通常唔係「畫得靚」，而係**幫你同學生睇清結構**——單元有幾條主線、概念點樣承接、考前溫習應該跟住邊條分支行。

## 幾時用 Mind Map？

**備課、科組備課**

- **新單元開局：** 快速睇晒「大概念 → 細概念」點樣掛鈎，方便你決定教學次序同板書結構。
- **跨文件比對：** 若果一個 notebook 入面有多份校本材料，Mind Map 可以當**結構式目錄**，之後你再逐段深入讀。
- **科組共用語言：** 用同一幅圖做討論起點，減少「大家講緊唔同層次」嘅情況。

**課堂／溫習**

- **考前總複習：** 用「概念關係」幫學生建立**心智模型**，而唔係淨係記散點。
- **專題／報告：** 學生整理背景資料時，可以用 Mind Map 做**第一層分類**，再由老師把關深度同例子。

**唔係萬能：** 若果你手上材料太散、或者目標係「逐句精讀」，可能先需要摘要或分段問答，再先至輪到 Mind Map。

## 實務流程

1. **先整理好來源**  
   同一個 notebook 盡量圍繞**同一個單元／議題**；來源愈聚焦，地圖愈有用。

2. **喺 Studio 揀 Mind Map**  
   生成後，花一兩分鐘睇**主幹同分支**係咪同你教學重點一致。你亦可以**逐個展開／收起分支**嚟瀏覽；介面入面好多時會有**一次過展開所有節點**嘅按鈕（或同類選項），方便你一眼睇晒成幅結構，先再決定下一步點下載或投影。若果想再試另一幅，一般要**刪除該則 Mind Map 筆記再生成**或**改來源**——詳見下方「老師常見問題」第一條。

3. **下載、列印同投影**  
   生成 Mind Map 之後，NotebookLM 一般可以將圖**匯出成圖片檔**。你可以：
   - **下載**後放入簡報或工作紙；
   - **列印**派發俾學生溫習、貼壁報，或者開科組會時每人有一張紙上手寫修訂；
   - 唔想派紙嘅班，亦可以**投影**全屏，全班一齊跟住分支講解。  
   咁樣就算堂上未必部部機都有得開網頁，仍然可以有一幅「結構總覽」跟住走。

4. **截圖作快速分享**  
   若果只係想即刻傳俾同事或家長群組，用系統截圖有時最快——但長遠備課建議仍用 NotebookLM **內建匯出**，解像度同裁切會較穩陣。

5. **瀏覽幅圖、跟住節點發問**  
   你可以**縮放、捲動**，以及**展開／收起分支**嚟瀏覽幅圖；亦可以**揀選地圖上嘅某一個節點**，喺聊天室**針對該個主題發問**（例如要澄清、要例子、要深淺唔同嘅講法）。

## 老師常見問題

**Q：可唔可以換一幅唔同結構嘅 Mind Map？**  
若果你想**換一幅唔同結構**，通常要 **刪除 Studio 裏面該則 Mind Map 筆記再重新生成**（More → Delete note 會重新走生成），或者**調整／增減來源**後再生成。無論哪種方式，**AI 只係建議**，最後要由你核實概念係咪同課程、校本一致。

**Q：Mind Map 同「音頻／影片概覽」有咩分工？**  
音頻／影片幫你**聽住吸收**；Mind Map 幫你**睇住結構**。可以先用 Mind Map 定方向，再按分支去聽細節。

**Q：學生會唔會「背圖唔背書」？**  
可以將 Mind Map 定位做**溫習地圖**，仍然要配合課本例子、題型同課堂講解——你喺堂上講清楚「點用幅圖」好重要。

**Q：香港網絡環境點用 NotebookLM？**  
若果你關心登入同地區限制，可參考我哋其他網誌（例如 VPN／Workspace 相關篇）；需要喺學校環境穩定使用，亦可了解 **[Magic Door 免費試用](/trial)**。

Mind Map 嘅價值，在於將**你揀過嘅來源**變成一幅**可討論、可修訂嘅概念圖**。老師把關準確度同教學設計，學生先有清晰嘅溫習路線。
