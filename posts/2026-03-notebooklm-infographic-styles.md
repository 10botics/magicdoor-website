Google NotebookLM 最新更新為資訊圖表功能帶來重大升級——新增了 **10 種預設視覺風格**，讓您將文件、筆記或研究資料一鍵轉化為精美的視覺摘要，完全不需要設計技能。

## 什麼是 NotebookLM 資訊圖表功能？

NotebookLM 的 Infographic 功能可以讀取您上傳的來源（PDF、文章、筆記），自動歸納要點，並生成一張帶有圖示、色塊和排版的靜態視覺摘要圖。

此次更新最大亮點，是新增了「**選擇視覺風格**」選項，讓圖表風格可以根據用途自由切換。

## 10 種預設風格一覽

| 風格 | 適合用途 |
|------|----------|
| **Professional**（專業） | 商務報告、家長信、行政文件 |
| **Instructional**（教學式） | 步驟指引、操作教程、學習流程 |
| **Scientific**（科學式） | 學術內容、研究摘要、數據展示 |
| **Editorial**（報章式） | 新聞式排版，帶復古報紙風格 |
| **Bento Grid**（便當格） | 結構化內容，清晰分區呈現 |
| **Bricks**（積木式） | 活潑動感，適合展示清單或步驟 |
| **Clay**（黏土感） | 色彩鮮豔，現代風格 |
| **Anime**（動漫風） | 角色驅動的創意視覺 |
| **Kawaii**（可愛風） | 輕鬆有趣，適合非正式場合 |
| **Sketch**（手繪風） | 手稿感設計，親切自然 |

## 如何使用？

1. 開啟 NotebookLM，上傳您的來源文件
2. 在右側「Studio」面板，點擊「Infographic」
3. 按鉛筆圖示進入自訂選項：
   - 輸入提示語（例如「用藍色主調，突出三個重點」）
   - 選擇圖表方向：直向、橫向或正方形
   - 選擇詳細程度：簡潔 / 標準 / 詳細
   - 選擇輸出語言
4. 生成完成後，可下載為 PNG 檔案或分享連結

## 老師可以怎樣用？

- **課前預習材料**：將一篇長文章轉成一張 Infographic，讓學生快速掌握重點
- **課堂總結**：把一節課的筆記變成視覺化摘要，方便溫習
- **家長通訊**：用「Professional」風格製作清晰的學校政策或活動簡介
- **跨科合作**：「Scientific」或「Instructional」風格適合 STEM 科目的教學圖表

## 注意事項

- 生成的圖表為**靜態圖片**，不能在 NotebookLM 內直接編輯文字
- 此功能**限 18 歲以上**用戶使用
- 正在**分階段全球推出**，部分用戶可能暫未見到此選項
- 涉及敏感話題或系統繁忙時，生成有機會失敗
